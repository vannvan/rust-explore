## v1.0.4 2023-12-05

### 使用说明

请采用`grd`(含义为group-resource-download)命令进行下载，即

> ytool grd

⚠️必需的参数 `yuque.config.json`

- host 团队空间域名(如：<https://xxxx.yuque.com>)
- skip 是否跳过同名文件

```json
{
   "username": "",
   "password": "",
   "host": "", 
   "skip": true 
}
```
